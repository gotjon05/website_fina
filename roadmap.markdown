## Roadmap for Grad School
Using this Roadmap to methodically prepare for gradschool in data science 

### Phase 1: Foundations (Current Focus)
- [Khan Academy - College Algebra](https://www.khanacademy.org/math/college-algebra)
- [DataCamp - Python for Data Science](https://www.datacamp.com/courses/intro-to-python-for-data-science)

---

### Phase 2: Advanced Math & Data Science
- [Khan Academy - Pre-Calculus](https://www.khanacademy.org/math/precalculus)
- [Khan Academy - Linear Algebra](https://www.khanacademy.org/math/linear-algebra)
- [Khan Academy - Probability & Statistics](https://www.khanacademy.org/math/statistics-probability)
- [DataCamp - Pandas & Data Visualization](https://www.datacamp.com/tracks/data-analyst-with-python)

---

### Phase 3: Machine Learning & Calculus
- [Khan Academy - Calculus 1](https://www.khanacademy.org/math/calculus-1)
- [Khan Academy - Calculus 2](https://www.khanacademy.org/math/calculus-2)
- [Andrew Ng - Machine Learning (Coursera)](https://www.coursera.org/learn/machine-learning)
- [ISLR - Introduction to Statistical Learning](https://www.statlearning.com/)

---

### Phase 4: Deep Learning & Advanced Applications
- [Khan Academy - Multivariable Calculus](https://www.khanacademy.org/math/multivariable-calculus)
- [Deep Learning Specialization - Andrew Ng](https://www.coursera.org/specializations/deep-learning)
- [Fast.ai - Practical Deep Learning](https://course.fast.ai/)
- [Google Cloud ML - Model Deployment](https://cloud.google.com/training/courses/machine-learning)
